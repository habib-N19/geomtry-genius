Q:What is DOM and what is the purpose of DOM?

Answer:

DOM(document object model) is a way of organizing every element of a website. We can compare it with a tree where every branch is a part of tree and every html element (heading, paragraph, form, input, etc) is a part of DOM model.

Purpose of DOM - dom is the way to tell the browser of the structure of a website. Dom makes sites more accessible. By using dom we can control every element of a website and interact with the website.

Q:How will you select HTML elements using DOM? Name the DOM methods.

Answer:

There are a various way of selecting and manipulating html elements using DOM of Javascript.Mainly 5 elements of dom selector available

  • getElementById()
  • getElementsByTagName()
  • getElementsByClassName()
  • querySelector()
  • querySelectorAll()

The method of using these selectors are given bellow: To use a selector we need to add the prefix of the document of which we are trying to select using the selector.

Methods are

  • document.getElementById()
  • document.getElementsByTagName()
  • document.getElementsByClassName()
  • document.querySelector()
  • document.querySelectorAll()

Here inside the parenthesis () we have to insert the id name or the relatable element name which we are trying to get.
Example: If we are trying to get an tag selector of h1 then the method is document.getElementsByTagName('h1')

Q:What is event bubble?

Answer:

Event bubble is a method of javascript DOM model to assign rules or conditions to multiple child element of a parent element.

We can compare it with a branch of a tree but this time it is going backwards. Meaning- Let's assume we have an unordered list. If we add an event listeners to the list items then it will propagate to the parent tag. If we click on the list item it also means we clicked on the list. If we have a div and inside that div if we have a button, then clicking button means also clicking or adding event listeners to the div element.

Point to be noted that if a parent element has multiple child element and we use event listeners to a particular child then the other child will not be triggered.

Q:What is a callback function and why will you use it?

Answer:

Javascript runs as interpreter language. Meaning it reads codes from top to bottom. Callback functions are used as an function argument to execute in later stage.Because as an asynchronous language , the interpreter will not stop for executing a function. To make some delay in the function we use callback function.It runs in the background while other code runs.

In simple form we can say if we have a function that just shows an output on the console after some time then we can make that function outside of the main function and then call that function as an argument in the main function

Here we can see we are executing the first function but it will be executed as we define inside the callback function.
