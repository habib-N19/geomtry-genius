Q-1 DOM(document object model) is a way of organizing every element of a website. We can compare it with a tree where every branch is a part of tree and every html element (heading, paragraph, form, input, etc) is a part of DOM model.

purpose of dom - dom is the way to tell the browser of the structure of a website. Dom makes sites more accessible. By using dom we can control every element of a website and interact with the website.
